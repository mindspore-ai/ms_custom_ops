# Open Source Attribution

本目录基于华为 Ascend 社区的开源项目 **CANN Ops Adv** ([https://gitee.com/ascend/cann-ops-adv](https://gitee.com/ascend/cann-ops-adv)) 进行二次开发与功能扩展。

主要基于原仓库中的 **FlashAttention** 算子逻辑进行修改和适配。

> 具体改动内容（新增 Sparse Attention Score 算子及相关优化）请查看相关目录下的详细 [README.md](../sparse_attention_score/README.md)。
